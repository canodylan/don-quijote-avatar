# Don Quijote Interactivo (Offline)

Aplicación web local sin backend para presentar un avatar de Don Quijote con voz (Text-to-Speech del navegador).

## Requisitos

- Navegador moderno (Chrome, Edge, Firefox o similar)
- Funciona sin internet una vez abierta

## Uso

1. Abre `index.html` en el navegador.
2. Escribe un texto y pulsa **Hablar**.
3. Ajusta la velocidad con el slider.
4. Usa frases rápidas para iniciar voz automática.
5. Pulsa **Pantalla completa** para modo presentación.

## Imágenes del avatar

Ubicación esperada:

- `img/quijote_0.png` (boca cerrada)
- `img/quijote_1.png` (boca semiabierta)
- `img/quijote_2.png` (boca abierta)

Se incluyen placeholders temporales que puedes reemplazar por tus imágenes definitivas.

## Notas

- Si no hay voz en español disponible, se usa la voz predeterminada del sistema.
- No requiere instalación de paquetes ni servidor.
